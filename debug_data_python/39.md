['A', 'A+', 'A-', 'AA', 'AA+', 'AA-', 'AAA', 'B', 'B-', 'BB', 'BB+', 'BBB', 'BBB+', 'BBB-', 'C', 'CC', 'CCC']

最终要求的结果

['A', 'A+', 'A-', 'AA', 'AA+', 'AA-', 'AAA', 'B', 'B-', 'BB', 'BB+', 'BBB', 'BBB+', 'BBB-', 'C', 'CC', 'CCC']
排序规则
1. A>B>C
2. AA+ > AA > AA- (这里的+-是我的问题根源所在)

### 修改方案：


```python
la = ['A', 'A+', 'A-', 'AA', 'AA+', 'AA-', 'AAA', 'B', 'B-', 'BB', 'BB+', 'BBB', 'BBB+', 'BBB-', 'C', 'CC', 'CCC']
def df(v):
    if v[-1]=="+":
        s = v[:-1]
        n = -1
    elif v[-1]=="-":
        s = v[:-1]
        n = 1
    else:
        s = v
        n = 0
    return (s,n)
la.sort(key=df)
print(la)
```
结果
['A+', 'A', 'A-', 'AA+', 'AA', 'AA-', 'AAA', 'B', 'B-', 'BB+', 'BB', 'BBB+', 'BBB', 'BBB-', 'C', 'CC', 'CCC']
