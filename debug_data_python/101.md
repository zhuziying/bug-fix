### 修改方案：
加一个break，登录成功就可以结束循环了。

```python

dic={'admin':'123456','cheer':'654321'}
def login():
 for name in dic:
    name=input("请输入账号：")
    if dic.__contains__(name):
        password=input('请输入密码：')
        if password==dic.get(name):
            print('登陆成功')
            break
        else:
            print('密码错误')
    else:
        print('账户名错误')
        zc()
 
def zc():
```
