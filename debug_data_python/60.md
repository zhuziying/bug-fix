### 问题描述：
<p>python计算器问题</p>
我写的计算器脚本：

```python
print('计算器')
while True:
    a=int(input('请输入第一个数字:'))
    b=str(input('请输入运算符号:'))
    c=int(input('请输入第二个数字:'))
    if b == '+':
        print('等于:',a+c)
    elif b == '-':
        print('等于:',a-c)
    elif b == '*':
        print('等于:',a*c)
    elif b == '/':
        print('等于:',a/c)
    else:
        print('错误')

```
只能计算两个数的结果，请问怎么改成多个数字运算的？ 
### 修改方案：
换个思路，轻松写意。

```python
>>> while True:

    if exp == '' or exp == 'bye':
        break
    try:

        print('算式错误，请重新输入。')
    if isinstance(result, int):
        print('%s = %d'%(exp, result))
    else:
        print('%s = %f'%(exp, result))
```
