### 修改方案：
你尝试一下将时间复杂度降下来试试,不调用第三方库操作,双重循环改为单循环

```python
len_as = input()
str_b = []
nummap = {}
for i in range(int(len_as)):
    str_b.append(input().split(' '))
for i in str_b:
    for j in range(int(i[0]),int(i[1])+1):
        if j in nummap:
            nummap[j] += 1
        else:
            nummap[j] = 1
nummap=sorted(nummap.items(),key=lambda x:x[0])
for k,v in nummap:
    print(f"{k} {v}")


```
