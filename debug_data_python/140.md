### 修改方案：


```python
str1=list(map(int,input().split()))
str2=str1[::2]
str1=str1[1::2]
res = []
for x in range(len(str1)):
    if str1[x]*str2[x]!=0:
        res.append(str(str1[x]*str2[x]))
        res.append(str(str1[x]-1))
if len(res)!=0:
    print(" ".join(res))
else:
    print(0,0)
 

```
